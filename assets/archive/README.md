# Archived unused assets

This folder stores **unused assets only** that are currently not loaded by the website.

## What is kept here

- **Legacy UI icons** – older menu, close, mail, RSS, footer, and arrow graphics
- **Unused favicons** – alternative favicon PNG files that are not currently linked in `index.html`
- **Archived company logos** – logos from the commercial projects section that are only present in commented-out markup
- **Alternative project screenshots** – older `Hammer Cash` images and an unused personal project screenshot (`fileUploader.jpg`)
- **Extra social media / logo variants** – older brand marks, transparent versions, and source artwork
- **Unused font files** – static `League Spartan` font weights that are not currently referenced by `style.css`

## Note

These files are kept for safekeeping and can be restored later if needed, but they are not part of the active site UI right now. If any asset becomes used again, it should be moved back out of `assets/archive`.
